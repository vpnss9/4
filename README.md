SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

Shadowrocket相关信息
一、Shadowrocket安卓版手机
目前，Shadowrocket没有官方的安卓版本。如果您在寻找类似的功能，可以考虑使用Shadowsocks、V2Ray等代理工具，这些工具在Android平台上有良好的支持。

二、Shadowrocket有电脑版吗
Shadowrocket是专为iOS设备设计的，官方并没有推出PC版。如果您希望在PC上使用类似功能，可以尝试使用其他代理工具，如Shadowsocks或V2Ray。

三、Shadowrocket账号密码怎么设置
在Shadowrocket中，您可以通过以下步骤设置账号和密码：

打开Shadowrocket：启动应用，进入主界面。
添加配置：在“配置”选项中，选择“添加配置”。
输入节点信息：在节点设置中，输入服务器地址、端口、加密方式以及账号和密码（如果需要）。
保存配置：设置完成后，记得保存配置。
四、Shadowrocket有安卓版本吗
如前所述，Shadowrocket并没有安卓版本。如果您需要类似的功能，可以寻找其他科学上网工具，如Shadowsocks或V2Ray，很多这类工具在Google Play Store中都有提供。

五、Shadowrocket节点超时怎么办
如果您在使用Shadowrocket时遇到节点超时的问题，可以尝试以下解决方案：

检查网络连接：确保设备的互联网连接正常。
更换节点：尝试使用其他代理节点，有时特定节点可能出现故障。
重启应用：关闭Shadowrocket并重新启动，检查连接状态。
确认配置：确保输入的节点信息（如地址、端口）正确无误。
六、Shadowrocket全局路由什么意思
全局路由是指所有的网络流量都通过代理服务器进行转发。选择全局路由模式后，您的设备在访问所有网站和应用时都会使用设定的代理，适合需要全面科学上网的用户。
